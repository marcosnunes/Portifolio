Projeto BestFood
Delivery

  • BstFood

A pedido de um amigo, desenvolvi um app de entrega de alimentos por pedidos online para dispositivos Android. Com o Android Studio para Windows, desenvolvi um app híbrido utilizando o recurso xWalkView. No Hostinger eu criei um site gratuito para testes e instalei o WordPress. No site, instalei o plugin GloriaFoods que oferece o recurso de pedido online de alimentos. Clique aqui para visualizar e baixar o SDK.

Aqui você pode baixar o APK que é o arquivo de instalação do app BestFood no seu smartphone Android. Para relizar a instalação você terá que habilidar a opção "Fontes desconhecidas" do seu aparelho. A localização desta opção varia de acordo com a marca do aparelho. Clique aqui para instalar.
